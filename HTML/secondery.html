Question-1: What are the different ways to select an element in the DOM?

Answer: The different ways to select an element in the DOM are getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML returns or sets HTML content, innerText returns or sets visible text (excluding hidden elements), and textContent returns or sets all text, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

Answer: Event delegation allows a parent element to handle events from its child elements using event bubbling, improving performance and handling dynamic elements efficiently.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a process where an event starts from the target element and propagates upward through its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: You can create elements with document.createElement(), add them with appendChild() or insertBefore(), and remove them with removeChild() or remove().
